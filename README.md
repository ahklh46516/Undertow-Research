# Undertow

**The macro layer for on-chain markets.**

Bitcoin, U.S. & Korean equities, gold, crude oil and the spot-crypto ETFs are not
independent — they breathe to the same rhythm: the expansion and contraction of global
dollar liquidity. Undertow measures that force. It estimates the unobserved *macro state*
of the world — a cross-asset risk regime and a composite liquidity index — from public
market data, and is designed to publish it on-chain as a verifiable primitive any
application can consume.

## The model (first-cut)

The engine pulls ~1 year of daily closes for BTC, ETH, S&P 500, Nasdaq, KOSPI, gold,
WTI crude, the dollar index (DXY) and the 10y Treasury yield from public market data, then
computes:

- **Conditional correlation** across the panel (trailing-window).
- **Macro Undertow Index** (0–100) — a composite of risk-asset momentum, dollar move,
  cross-asset correlation and realized volatility, percentile-normalized.
- **Risk regime** — Risk-On / Neutral / Risk-Off — with state probabilities.
- **Per-asset beta** to the index, a transition matrix, and regime statistics.

## Run locally

```bash
# 1. compute a fresh snapshot
python build_data.py

# 2. serve the site
python -m http.server 8777

# 3. open
#    http://localhost:8777/index.html      (whitepaper)
#    http://localhost:8777/terminal.html   (dashboard)

# optional: keep data fresh (re-runs the engine every ~2 min)
python live_update.py
```

The dashboard polls `data.json` every 45s, so it updates without a manual refresh.

Requires Python 3 with `pandas` and `numpy`. Data source: public market data
(Yahoo Finance chart API). No API key needed.

## Status & disclaimers

This is a **research project under active development**. The model describes statistical
relationships, not certainties: *correlation is not causation, and past behavior does not
guarantee future results.* The regime/index are a first-cut model; the on-chain oracle is
**not yet deployed**. Nothing here is financial, investment, or legal advice, nor an offer
to buy or sell any security or token.
